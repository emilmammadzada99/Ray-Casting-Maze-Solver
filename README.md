# 2D Maze Light Animation

A 2D maze game made with Python and Pygame featuring real-time ray casting, dynamic lighting, and a BFS automatic maze solver.

## Features

- Real-time light simulation
- Random maze generation
- BFS auto solver
- Compass guidance system
- Mouse-controlled movement
- Wall collision detection
- Dynamic shadows and glow effects

## Controls

- Mouse → Move lamp
- A → Toggle auto solver
- R → Generate new maze
- ESC → Exit
